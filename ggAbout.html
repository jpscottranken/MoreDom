In a typical number guessing game, the user inputs a number between 1 and 100. Checks are made to assure that the number inputted is within range. If it is, it is counted as a "valid guess". If not, an error message is given.

If the user guess is too low or too high, an associated message is given, and the total number of guesses is incremented. If the right number is eventually guessed, the user is told so and is "rated" as Expert (1 - 5 guesses), Average (6 - 10 guesses), or Novice (more than 10 guesses made).
